# Programming-Langauages-Enum-Projest(Rust/Python)
Project Overview

Python and Rust enum analysis

This repository contains a comparative study of enumeration (enum) implementations in Rust and Python. The goal is to investigate the languages’ type systems, behavior, and design choices when working with enums, including normal usage, edge cases, and beyond-normal usage scenarios.

This project was completed as a group assignment for [Your Course/Professor Name], demonstrating:

Basic enum declaration, assignment, and printing Exhaustive control flow with enums Extended enum behavior with associated data Iteration and introspection Applied enumeration in real-world scenarios Comparative analysis between Rust and Python
